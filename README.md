## setting up mandrill in Rails~4

#### app includes
- user login confirmation
- unlock instructions
- forgot password
- feedback page
- contact-me page

#### get this app

```
git clone git@github.com:rinoldsimon/contact-app.git
cd contact-app
bundle install
rake db:migrate
rails server
```
Switch to `localhost:3000` in your browser
